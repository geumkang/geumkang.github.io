4) First-order Taylor approximation at the Point A

In function f(x), we can approximate f(x) about the point where x = a by the polynomial $$ f(x) = f(a) + f'(a)(x-a) + \dfrac{f''(a)}{2!}(x-a)^2 + \cdots + \dfrac{f^{(n)}(a)}{n!}(x-a)^n $$

If we want to approximate this to first order, it means $$ \hat{f}(x) \approx f(a) + f'(a)(x-a) $$

In this case, The x coordinates of Point A is 50. (x = 50) $$ \hat{f}(x) \approx f(50) + f'(50)(x-50)\\ f'(x) = \frac{\partial f}{\partial x}(x^3 - {1 \over 2}x^2 + 4x) = 3x^2 - x + 4\\ f'(50) = 7454\\ \therefore \hat{f}(x) \approx 7454x - 247850 $$
